# 4-Bit Ripple Carry Adder

## Objective
To design and implement a 4-bit Ripple Carry Adder using Verilog HDL.

## Description
A Ripple Carry Adder is built using four full adders connected in series. The carry output of one full adder becomes the carry input of the next stage.

## Software Used
- Verilog HDL
- ModelSim / Vivado

## Files Included
- ripple_carry_adder.v
- ripple_tb.v
- output.png

## Result
The 4-bit Ripple Carry Adder was designed and verified successfully using simulation.
